# Dataset Card

## Назва
NLPholybnycha Lab 2 dataset – експериментальні дані меж зерен

## Очищення
- raw/: 1 файлів оригінальних даних
- processed_v2/: 1 очищених та нормалізованих файлів
- sample/: 1 файлів для швидкого тестування
- Видалено дублікатні та пропущені записи
- Проведено нормалізацію числових ознак

## Ризики
- Можливий шум у вимірюваннях
- Неповні записи у raw даних
- Дані можуть містити зміщення через специфіку експерименту
